# 03 Tailwind Props Project

##  What I Learned
This project is focused on reusable components and styling.

I learned:
- Props in React
- Component reusability
- Tailwind CSS styling
- Default prop values
- Passing data between components

##  Project
A reusable Card component that displays:
- Username
- Button text
- Styled UI using Tailwind CSS

##  Key Concept
Props allow us to pass data into components and reuse the same component with different content.

##  Example Usage
<Card username="Hassan" btnText="Visit Profile" />

##  How to Run
npm install
npm run dev
