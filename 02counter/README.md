# 02 Counter App

##  What I Learned
In this project, I learned about React state.

I learned:
- useState hook
- Event handling (onClick)
- Updating UI dynamically
- Re-rendering in React

##  Project
A simple counter application with:
- Increment button
- Decrement button
- Reset button

##  Key Concept
React state allows UI to change dynamically without reloading the page.

##  How to Run
npm install
npm run dev
